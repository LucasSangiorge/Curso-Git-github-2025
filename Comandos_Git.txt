Comandos Básicos para Git
Arquivo novo
git init . -> Cria um repositorio git na pasta atual
git status -> Mostra o status do trabalho de cada arquivo
gitt add nome_arquivo -> adciona arquivos para ser commit
git commit -m "msg" -> Faz o commit do que estava para ser commitado

pwd -> Mostra o diretorio onde estamos.
ls -> Mostra os arquivos dentro da pasta
ls -a -> Mostra arquivos oculto
cd -> Muda o diretorio
cd .. -> Volta um diretorio
mkdir -> Cria uma nova pasta onde está o diretório
rm -> Remove arquivo ou diretórios
rm -rf -> Remove de forma recursiva e forçada todos diretorios

git log -> Mostra todos os históricos de todos os nossos saves, dos mais atuais para os mais antigos
