Comandos Básicos para Git
Arquivo novo
git init . -> Cria um repositorio git na pasta atual
git status -> Mostra o status do trabalho de cada arquivo
gitt add nome_arquivo -> adciona arquivos para ser commit
git commit -m "msg" -> Faz o commit do que estava para ser commitado

pwd -> Mostra o diretorio onde estamos.
ls -> Mostra os arquivos dentro da pasta
ls -a -> Mostra arquivos oculto
cd -> Muda o diretório
cd .. -> Volta um diretório
mkdir -> Cria uma nova pasta onde está o diretório
rm -> Remove arquivo ou diretórios
rm -rf -> Remove de forma recursiva e forçada todos diretórios

git log -> Mostra todos os históricos de todos os nossos saves, dos mais atuais para os mais antigos

git add . -> O ponto quer dizer que add  arquivos do diretório atual
git add * -> Add tudo do diretório atual
git diff nome do arquivo -> Mostra o que foi trabalhado no dia anterior o que foi modificado e não commitamos
git diff . -> Mostra o que foi modificado em todos os arquivos.
git checkout nome da branch -> ele cria uma novo arquivo e estamos trabalhando em um novo arquivo.
git merge 'nome' -> Ela junta o commit secundário e joga para o meio da principal para ele não sumir.
git branch -d -> Ele deleta a branch

--Fluxo de trabalho Git
1.Criar o diretório do projeto
2.iniciar o git dentro do diretório criado (got init.)
3.editar ou criar arquivos
4.git status
5.git add <nome arquivos>
6.git status
7.git commit -m "Mensagem"
8.git status
9.Volta para o passo 3
